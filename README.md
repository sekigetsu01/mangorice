# Mangorice

## Installation:

```
git clone https://github.com/sekigetsu01/mangorice.git
cd ~/mangorice
```

Alternatively, you could use [SWAI](https://github.com/sekigetsu01/SWAI.sh).
This latter option is preferable.

## What does this setup achieve?

This configuration creates a fully functional linux workflow. Software is all foss, minimal
and typically cli based. This not only improves privacy and security, but it means that
things can run quickly and efficiently on your device. Additionally, this setup prioritizes
ergonomics, this is demonstrated by the vim centric keybinds and vim based applications. The
Window manager used is mangowm, this is similar to dwm but more modern and easily customizable.

## Customization

Given the minimality, it is simple to customize everything. configuration files are located
in `~/.config` and scripts are located in `~/.local/bin`. The `.zshrc` however, is located in
the home directory.

## Software used

Browser - Librewolf

Email Client - Thunderbird

Screenshot Tool - Flameshot

Text Editor - Neovim

Image Viewer - Swayimg
